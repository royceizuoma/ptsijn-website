# PTSIJN Foundation Website

This project is a Vite + React website for the Perfecting The Saints IJN Foundation.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This repository is set up to deploy the built site to GitHub Pages.

1. Push the repository to GitHub.
2. In GitHub, open the repository Settings → Pages.
3. Choose GitHub Actions as the source.
4. The workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml) will publish the site automatically.
